
# MoodMusic Magic Box 🌦️🎵

A dynamic web app that recommends music playlists based on **real-time weather** and **user mood**.

## ✨ Features
- Weather detection using Open-Meteo API  
- Mood-based Spotify playlist recommendations  
- Interactive UI with animated transitions  
- Responsive design (works on mobile/desktop)  

## 🛠️ Tech Stack
- **Frontend**: HTML5, CSS3, JavaScript  
- **APIs**: Open-Meteo, Spotify (via curated playlists)
